b1gMail Server Test Readme

Thank you for your interest in b1gMail! Using this Server Test script you can check wether b1gMail is compatible with your webspace/server or not and if your server configuration limits some of b1gMail's functionality.

Instructions

Please use the following instructions to test your server:
  1. Unzip, if not yet done, the file "servertest.zip".
  2. Using a FTP client or a file transfer application of your choice, upload the file "servertest.php" to your webspace.
  3. Open the uploaded file "servertest.php" in your web browser.
  4. You should see the test results. In case your webspace fails to meet some requirements, please ask your webhoster to enable the missing features.

Help
